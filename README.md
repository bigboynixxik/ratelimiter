# Rate Limiter для асинхронных запросов

### Ограничитель частоты запросов к API, реализованный на основе алгоритма Скользящее окно (Sliding Window). 

Гарантирует строгое соблюдение лимитов (например, ровно 10 запросов в 1 секунду) при полностью асинхронной работе, предотвращая получение ошибки HTTP 429 (Too Many Requests).

## Структура проекта

- `src/main.py` — Класс ограничителя SlidingWindowRateLimiter.

- `src/test.py` — Тестовый скрипт, имитирующий 100 одновременных запросов к API.

## Инструкция по запуску

Откройте терминал в папке с проектом и выполните следующие шаги:

### 1. Создание виртуального окружения

Создайте изолированное окружение с помощью команды:

```bash
python -m venv venv
```

### 2. Активация виртуального окружения

Активируйте окружение в зависимости от вашей операционной системы:

**Для Windows:**

```bash
venv\Scripts\activate
```

**Для macOS / Linux:**
```bash
source venv/bin/activate
```

### 3. Запуск тестирования

```bash
python src/test.py
```
